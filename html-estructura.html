5. Introducción a la Programación: Algoritmos y Estructuras en Python

31 de mayo de 2026 | Ciencias de la Computación

Programar consiste en dar instrucciones claras y ordenadas a una computadora para resolver un problema. Python es un lenguaje de alto nivel, interpretado y multiparadigma, diseñado bajo una filosofía clara: hacer que el código sea sencillo, legible y accesible.

Flujo de Ejecución en un Lenguaje Interpretado
Código Fuente
script.py (Legible)
Intérprete

Ejecuta la acción
Esquema 1: Python no necesita compilación previa completa; el intérprete procesa el script directamente en tiempo de ejecución.

Características Principales del Lenguaje

Creado por Guido van Rossum en 1991, Python se destaca en el desarrollo de software moderno gracias a una serie de propiedades estructurales clave:

  • Sintaxis Limpia y Legible: El código parece pseudocódigo en inglés. Para delimitar bloques de funciones o condicionales no se usan llaves `{ }`, sino la **indentación** (espaciado obligatorio), lo que fuerza a mantener el orden de forma nativa.
  • Tipado Dinámico: No es necesario declarar explícitamente si una variable es de tipo entero (`int`), texto (`string`) o decimal (`float`). Python deduce automáticamente el tipo de dato según el valor asignado.
  • Multiplataforma y Multiparadigma: Un mismo archivo funciona en Windows, Linux o macOS. Además, soporta programación imperativa, funcional y orientada a objetos de manera flexible.

Lógica y Control de Flujo: Condicionales

Las computadoras toman decisiones lógicas evaluando expresiones booleanas. En Python, esto se maneja mediante las estructuras de control condicional (`if`, `elif`, `else`) y la estructura de emparejamiento estructural (`match case`):

# Ejemplo de toma de decisiones en Python
nota = int(input("Ingrese la nota final: "))

if nota >= 60:
    print("Suficiente - Aprobado")
else:
    print("Reprobado")

El bloque anterior evalúa el valor numérico ingresado y altera el camino del flujo lógico según la condición establecida, bifurcando el resultado de la salida.

Bloques Estructurales de un Algoritmo

Todo desarrollo algorítmico básico combina tres operaciones fundamentales en su arquitectura:

Entradas (Variables)
Captura de datos mediante variables con asignación dinámica o teclado.
Procesos (Operadores)
Cómputo aritmético, manipulación de cadenas de texto y comparaciones lógicas.
Salidas (Funciones)
Despliegue de los resultados procesados en pantalla utilizando la función `print()`.

¿Por qué elegí este tema y dónde se aplica?

Mi perspectiva: Estudiar lógica de programación y algoritmos expande la capacidad analítica para resolver problemas complejos de forma secuencial. Python, gracias a su facilidad de lectura, permite centrar toda la atención en diseñar soluciones eficientes sin que la sintaxis o reglas pesadas se conviertan en un obstáculo para la abstracción lógica.

¿Dónde se aplica? Se aplica directamente en el desarrollo de software e ingeniería de sistemas, automatización de scripts e infraestructura en la nube, optimización de consultas avanzadas, machine learning, ciencia de datos y backend de plataformas web.
